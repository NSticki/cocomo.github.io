# Веб-калькулятор оценки затрат на разработку ПО по моделям COCOMO 81 и COCOMO II

##  Описание проекта
Данный программный продукт представляет собой универсальный веб-калькулятор, реализующий математический аппарат конструктивных моделей стоимости **COCOMO 81** и COCOMO II (разработанных Барри Боэмом). Программа автоматизирует процесс расчета ключевых экономических показателей ИТ-проектов, включая трудоемкость, календарные сроки разработки и оптимальную численность проектной команды.

Проект выполнен в рамках лабораторной работы по дисциплине «Программная инженерия».

---------------------

## Функциональные возможности
Программный комплекс поддерживает расчет по 4 взаимосвязанным моделям, описанным в методических указаниях:
1. COCOMO 81 (Basic / Базовый уровень): Быстрая оценка на основе объема кода (`SIZE` в KSLOC) для трех типов проектов: Organic (Распространенный), Semidetached (Полунезависимый), Embedded (Встроенный).
2. COCOMO 81 (Intermediate / Промежуточный уровень): Расширенная оценка с учетом поправочного мультипликатора стоимости `EAF`, формируемого на основе 15 факторов затрат (атрибутов ПО, аппаратной платформы, персонала и проекта).
3. COCOMO II (Early Design / Стадия раннего проектирования): Оценка на начальных этапах жизненного цикла с использованием 5 факторов масштаба (`SF`), формирующих нелинейную экспоненту `E`, и 7 укрупненных множителей трудоемкости.
4. COCOMO II (Post-Architecture / Пост-архитектурная стадия): Детальная оценка после проектирования архитектуры с использованием 17 точных факторов затрат (`Cost Drivers`).
